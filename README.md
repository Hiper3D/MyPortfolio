# 💻 Personal Portfolio: Source Code

This repository contains the raw HTML and CSS files for my personal portfolio website. 

### ☁️ Cloud Hosting & Architecture
I do not use basic shared hosting for this project. This application is securely deployed on a custom, production-grade **AWS [Amazon Web Services - a comprehensive cloud computing platform]** architecture using **Amazon S3 [Simple Storage Service - an object storage service provided by Amazon for storing files]**, **CloudFront [Content Delivery Network - a geographically distributed network of servers to deliver content faster]**, and **OAC [Origin Access Control - a security feature that restricts access to your storage so only your distribution can read it]**.

🔗 **[Click here to view the complete AWS Infrastructure Documentation and Live Secure URL [Uniform Resource Locator - the standard web address used to find a specific page or file on the internet]](https://github.com/Hiper3D/aws-S3-static-hosting)**
